## 10.3Â Configuring Hierarchical States

Hierarchical states can be defined by using multiple`withStates()`calls where`parent()`can be used to indicate that these particular states are sub-states of some other state.

```
@Configuration
@EnableStateMachine
public
class
 Config2
        
extends
 EnumStateMachineConfigurerAdapter
<
States, Events
>
 {

    
@Override
public
void
 configure(StateMachineStateConfigurer
<
States, Events
>
 states)
            
throws
 Exception {
        states
            .withStates()
                .initial(States.S1)
                .state(States.S1)
                .and()
                .withStates()
                    .parent(States.S1)
                    .initial(States.S2)
                    .state(States.S2);
    }

}
```
